## Get started

* Clone the repository

    ```
    git@github.com:Helgen97/ugcc-project.git
    ```
* Open project in your IDE.
* Run
  ```
  mvn clean
  mvn spring-boot:run
  ```
* After application starts, go to ``http://localhost:8080``
* Enjoy
