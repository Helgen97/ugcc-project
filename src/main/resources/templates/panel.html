Виберіть пункт з меню для початку роботи з сайтом

Важливо для ознайомлення!

Перед тим, як додавати щось на сайт - прочитайте дану інформацію. 

  1. При завантаженні фотографії або файлу, переконайтесь, що їх розмір не перевищує 50МБ.
  2. Для швидкого завантаження картинки на сайті, дуже бажано, щоб розміри картинки були приблизно 600х500. Це покращить швидке завантаження сторінок на усіх пристроях. Надаю декілька сервісів, для змінювання розмірів зображень.
    • https://photo-resizer.ru
    • https://www.iloveimg.com/ru/resize-image
  3. Також рекомендую зжимати розмір фото, перед завантаженням. Надаю декілька безкоштовних ресурсів, для цього.
    • https://tinyjpg.com
    • https://tinypng.com
    • https://www.iloveimg.com/ru/compress-image
  4. При додаванні трансляції, у полі вводу розкладу, важливо, писати розклад з нового рядка. Під полем вводу є підказка, ознайомтесь, будь ласка. 
  5. При додаванні тексту до новин, статей є вкладка додати зображення. Там є пункт, де можна вибрати файл з комп'ютера. Не бажано і не рекомендую користуватися. Використовується багато символів для цього. Краще додайте зображення через посилання. 
  6. Для того, щоб додати відео в альбом: на сторінці з відео
    • Натисніть поділитися - вставити - скопіюйте код для вставки і вставте його у поле вводу.
  7. При появі помилок при створенні або завантаженні будь-чого: 
    1. Перезавантажте сторінку, та спробуйте ще раз. 
    2. Якщо проблема не зникла - вийдіть з акаунту, закрийте сторінку, і зайдіть знову і спробуйте додати ще раз. 
    3. Якщо все ж таки не працює, відкрийте консоль у браузері(права кнопка миші по сторінці - перевірити об'єкт - консоль), скопіюйте помилку(виділено червоним), та напишіть мені - я допоможу.
